Helpline

Citizen support channels for the Principality of Aetheria. If you need help, start with the right channel to get a faster, more useful response.

Support Channels

Citizen Services

Account access, credentials, onboarding, and status questions. Use this channel if you are new or returning and need navigation help.

Technical Support

Website issues, tool errors, integration problems, and accessibility feedback. Include the tool name and any steps you already tried.

Governance

Assembly correspondence, procedural questions, petition follow-up, and protocol clarifications for active participants.

Contact Options

Public Message Board

The primary support surface for written requests and status updates. Use it whenever your request is not time-sensitive or secret.

Collaboration Channels

Real-time coordination is handled through the governance and operations coordination channels linked from Citizen Hub and engagement pages.

Formal Correspondence

Use the contact channels for press, partnerships, or formal governance replies. Label the subject line so triage routes correctly.

Response-Time Expectations

AETHERIA operates with distributed contributors and limited administrative capacity. Typical response time depends on channel and workload, but you should expect:

  • Routine requests: a few days under normal conditions.
  • High-traffic periods: slower responses are more likely; early contact helps.
  • Clear requests: always receive faster handling than vague or broad inquiries.

If your request is urgent, say so explicitly and reference any open governance or ticket thread. If you are new and unsure where to start, visit Getting Started for a guided path.

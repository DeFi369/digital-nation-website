# Digital Nation Graphic Assets

Dark-themed SVG motif set for the public website.

## Files

- `assets/images/digital-city-skyline.svg`
  Panoramic header/background motif. Use as a full-width background image on hero sections.

- `assets/images/digital-node-network.svg`
  Central hub-and-spoke network motif. Good for feature imagery, about-page visuals, or large accent graphics.

- `assets/images/digital-nation-logo.svg`
  Wordmark-style logo asset. Can be used for branding, headers, or footer identity graphics.

- `assets/images/digital-node-header.svg`
  Minimal bar-and-node repeated header strip. Works as a subtle page header or section divider on dark pages.

## Usage

All assets are pure SVG and scale without quality loss. Example HTML usage:

```html
<img src="/assets/images/digital-city-skyline.svg" alt="" aria-hidden="true">
```

Or as a CSS background:

```css
.hero {
  background-image: url('/assets/images/digital-city-skyline.svg');
  background-size: cover;
  background-position: center;
}
```

## Design Notes

- Dark navy/black base with cyan/white neon accents.
- Compatible with a dark public-facing site theme.
- No included text unless noted; logo SVG includes the brand wordmark.
